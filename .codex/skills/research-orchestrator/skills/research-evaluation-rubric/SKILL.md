# Research Evaluation Rubric

Use this skill when the user asks for a quality judgment, rubric-style assessment, student-note evaluation, or scored review rather than ordinary research continuation.

## Read Next

- `references/evaluation-rubric-contract.md`
- `references/user-visible-language-contract.md`

## Operating Rules

- inspect the material before judging it
- evaluate against explicit criteria rather than vibe
- if you score something, explain why with evidence
- tie strengths and weaknesses to concrete text or archive artifacts when possible
- keep missing evidence distinct from weak evidence
- if the material is too thin for a confident score, say so plainly
- end with the one or two revisions that would most improve the work
