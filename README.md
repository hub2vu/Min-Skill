# Min-Skill

Personal Codex skill collection.

## Skills

- `skills/pro`: Runs Oracle browser mode from PowerShell to consult ChatGPT Pro GPT-5.5 Pro through a logged-in ChatGPT browser session.
- `skills/goal-agent`: Runs `/goal` requests through a goals-enabled Codex CLI child agent and stores run artifacts under the active workspace.
- `skills/research-orchestrator`: Bundles Research-Orchestrator as a file-based research workflow archive with its internal skills and references preserved.

## Layout

Each skill lives in its own folder under `skills/<skill-name>/` so the repository can grow without mixing unrelated skill resources.
