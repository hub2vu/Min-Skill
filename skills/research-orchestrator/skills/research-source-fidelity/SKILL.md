# Research Source Fidelity

Use this skill whenever the answer cites or links to sources, standards, papers, repositories, or documentation.

## Read Next

- `references/source-fidelity.md`

## Operating Rules

- prefer direct originals, canonical DOI landings, official documentation, official repository pages, and primary papers
- do not substitute search pages, generic homepages, or derivative summaries when a better direct target exists
- cite only inspected and verified sources
- when a canonical source has multiple legitimate homes, choose one defensible direct target and note the boundary only if it matters
- when source precision is not yet fixed, ask the smallest clarifying question needed before linking
- keep factual claim width matched to what the inspected source actually supports

## Output Rules

- show clear links
- keep attribution explicit
- separate verified fact from inference, memory, and open questions

## Stop Rule

If you cannot verify a needed source, say so plainly rather than fabricating or padding with weaker substitutes.
